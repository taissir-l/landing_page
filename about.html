About Frogy on Quest

Fun game about frogs and quests

"Frogy on Quest" is an engaging and challenging platformer game developed using Python and the Pygame library. Inspired by classic platformers, the game features a charming frog character embarking on a quest through various terrains filled with obstacles, traps, and enemies. The player must navigate these terrains using intuitive controls and mechanics such as jumping and double jumping.
